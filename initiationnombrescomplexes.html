Chapitre 1

Initiation aux Nombres Complexes

Les nombres complexes constituent une extension de l'ensemble des nombres réels. Ils permettent de résoudre des équations qui n'admettent aucune solution dans \(\mathbb{R}\) et jouent un rôle essentiel dans l'algèbre, la géométrie et la physique.

I. Pourquoi introduire les nombres complexes ?

Situation problème

Considérons l'équation :

\[ x^2+3x-4=0 \]

Son discriminant est :

\[ \Delta=3^2-4\times1\times(-4)=25 \]

Cette équation possède deux solutions réelles :

\[ x_1=1 \qquad ; \qquad x_2=-4 \]

Considérons maintenant :

\[ x^2+3x+4=0 \]

Son discriminant est :

\[ \Delta=3^2-4\times1\times4=-7 \]

Le discriminant est négatif. Aucun nombre réel n'a pour carré \(-7\).

Il devient donc nécessaire d'introduire un nouvel ensemble de nombres.

II. L'unité imaginaire

Définition

On introduit un nouveau nombre noté \(i\), appelé unité imaginaire.

\[ i^2=-1 \]

Ce nombre n'appartient pas à l'ensemble des nombres réels.

\[ i\notin\mathbb{R} \]
Exemple
\[ \sqrt{-7}=i\sqrt7 \]

III. Forme algébrique d'un nombre complexe

Définition

Tout nombre complexe s'écrit sous la forme :

\[ z=a+bi \]

où \(a\) et \(b\) sont deux nombres réels.

Partie réelle et partie imaginaire

Pour \(z=a+bi\) :

\[ \operatorname{Re}(z)=a \] \[ \operatorname{Im}(z)=b \]
Exemples
Nombre complexe Partie réelle Partie imaginaire
\(3+2i\) 3 2
\(-5+i\) -5 1
\(4\) 4 0
\(-7i\) 0 -7
À retenir

Un nombre complexe est réel si sa partie imaginaire est nulle.

Un nombre complexe est imaginaire pur si sa partie réelle est nulle.

\[ \mathbb{R}\subset\mathbb{C} \]

IV. Calculs dans \(\mathbb C\)

Addition
\[ (a+bi)+(c+di) = (a+c)+(b+d)i \]
Multiplication
\[ (a+bi)(c+di) = (ac-bd)+(ad+bc)i \]
Exemple
\[ (2-3i)(4+5i) \] \[ =8+10i-12i+15 \] \[ =23-2i \]
Identités remarquables
\[ (a+ib)^2 = a^2-b^2+2abi \]

\[ (a-ib)^2 = a^2-b^2-2abi \]

\[ (a+ib)(a-ib) = a^2+b^2 \]
Mise sous forme algébrique
\[ z= \frac{2+3i}{3-4i} \]

On multiplie par le conjugué du dénominateur.

\[ z= \frac{(2+3i)(3+4i)} {(3-4i)(3+4i)} \] \[ z= \frac{-6+17i}{25} \] \[ z= -\frac6{25} + \frac{17}{25}i \]

V. Représentation géométrique des nombres complexes

Affixe d'un point

Dans un repère orthonormé \((O;\vec i,\vec j)\), à tout point \(M(a,b)\), on associe le nombre complexe :

\[ z_M=a+ib \]

Ce nombre complexe est appelé affixe du point \(M\).

À retenir

Un point possède une position unique dans le plan. Son affixe permet de repérer cette position.

VI. Affixe d'un vecteur

Définition

Soit le vecteur :

\[ \vec u= \begin{pmatrix} a\\ b \end{pmatrix} \]

Son affixe est :

\[ \operatorname{aff}(\vec u)=a+ib \]
Observation

Ces deux vecteurs ont la même direction, le même sens et la même longueur.

Ils représentent donc le même vecteur et possèdent la même affixe.

⚠️ Ne pas confondre

Affixe d'un point et affixe d'un vecteur
Affixe d'un point Affixe d'un vecteur
\(z_M=a+ib\) \(\operatorname{aff}(\vec u)=a+ib\)
Le point est fixé. Le vecteur est libre.
Une seule position. Peut être déplacé parallèlement.

Même si les écritures se ressemblent, elles ne représentent pas le même objet mathématique.

VII. Affixe du vecteur \(\overrightarrow{AB}\)

Propriété fondamentale

Soient :

\[ z_A=a+ib \] \[ z_B=c+id \]

Alors :

\[ \boxed{ \operatorname{aff}(\overrightarrow{AB}) = z_B-z_A } \]
Justification
\[ z_B-z_A =(c+id)-(a+ib) \] \[ =(c-a)+i(d-b) \]

Or les coordonnées du vecteur \(\overrightarrow{AB}\) sont :

\[ \begin{pmatrix} c-a\\ d-b \end{pmatrix} \]

On retrouve donc exactement son affixe.

VIII. Milieu d'un segment

Propriété

Si \(I\) est le milieu du segment \([AB]\), alors :

\[ \boxed{ z_I= \frac{z_A+z_B}{2} } \]

IX. Propriétés des affixes

Multiplication par un réel
\[ \boxed{ \operatorname{aff}(\alpha\vec u) = \alpha\operatorname{aff}(\vec u) } \]
Somme de deux vecteurs
\[ \boxed{ \operatorname{aff}(\vec u+\vec v) = \operatorname{aff}(\vec u) + \operatorname{aff}(\vec v) } \]

X. Application : Centre de gravité d'un triangle

Théorème

Si \(G\) est le centre de gravité du triangle \(ABC\), alors :

\[ \overrightarrow{OG} = \frac13 \left( \overrightarrow{OA} + \overrightarrow{OB} + \overrightarrow{OC} \right) \]

En écriture complexe :

\[ \boxed{ z_G= \frac{z_A+z_B+z_C}{3} } \]
Exemple guidé

On considère :

\[ z_A=1+i \] \[ z_B=3+6i \] \[ z_C=5+2i \]

Déterminer l'affixe du centre de gravité \(G\).

Méthode 1
\[ z_G= \frac{z_A+z_B+z_C}{3} \] \[ z_G= \frac{ (1+i) +(3+6i) +(5+2i) }{3} \] \[ z_G= \frac{9+9i}{3} \] \[ \boxed{ z_G=3+3i } \]
Méthode 2 (à partir du milieu)

Le milieu \(I\) du segment \([BC]\) vérifie :

\[ z_I= \frac{z_B+z_C}{2} \] \[ z_I= \frac{(3+6i)+(5+2i)}{2} \] \[ z_I=4+4i \]

Comme :

\[ \overrightarrow{AG} = \frac23 \overrightarrow{AI} \] on obtient : \[ z_G-z_A = \frac23 (z_I-z_A) \] \[ z_G-(1+i) = \frac23[(4+4i)-(1+i)] \] \[ z_G-(1+i) = \frac23(3+3i) \] \[ z_G-(1+i) = 2+2i \] \[ \boxed{ z_G=3+3i } \]
Synthèse du chapitre
  • \(i^2=-1\)
  • \(z=a+bi\)
  • \(\operatorname{Re}(z)=a\)
  • \(\operatorname{Im}(z)=b\)
  • \(\operatorname{aff}(\overrightarrow{AB})=z_B-z_A\)
  • \(z_I=\dfrac{z_A+z_B}{2}\)
  • \(z_G=\dfrac{z_A+z_B+z_C}{3}\)

X bis. Distance entre deux points

Distance entre deux points

Soient deux points \(A\) et \(B\) d'affixes respectives \(z_A=a+ib\) et \(z_B=c+id\).

La distance \(AB\) est donnée par :

\[ AB= \sqrt{ (c-a)^2+(d-b)^2 } \]
Exemple

Soient :

\[ z_A=1+i \] \[ z_B=5+4i \]

Alors :

\[ AB = \sqrt{ (5-1)^2+(4-1)^2 } \] \[ AB = \sqrt{16+9} \] \[ AB=5 \]

XI. Longueur d'un vecteur

Norme d'un vecteur

Soit :

\[ \vec u= \begin{pmatrix} a\\ b \end{pmatrix} \]

La longueur du vecteur \(\vec u\) est :

\[ ||\vec u|| = \sqrt{a^2+b^2} \]
Lien avec les affixes

Si :

\[ \operatorname{aff}(\vec u)=a+ib \]

alors la longueur du vecteur est obtenue à partir des coordonnées de son affixe.

\[ ||\vec u|| = \sqrt{ (\operatorname{Re}(z))^2 + (\operatorname{Im}(z))^2 } \]

XII. Activité préparatoire

À faire

Calculer les longueurs :

\[ OA \] \[ OB \] \[ OC \]

où :

\[ A(1,0) \] \[ B \left( -\frac12, \frac{\sqrt3}{2} \right) \] \[ C \left( -\frac12, -\frac{\sqrt3}{2} \right) \]
Observation

Les trois points sont situés sur un même cercle de centre \(O\).

Cette configuration apparaîtra dans l'étude du nombre complexe \(j\).

XI. Compléments géométriques utiles

Triangle équilatéral

Un triangle est dit équilatéral lorsque ses trois côtés ont la même longueur.

\[ AB=BC=CA \]

Dans un triangle équilatéral :

  • Les trois angles mesurent \(60^\circ\).
  • Les médianes sont aussi des hauteurs.
  • Les médianes sont aussi des médiatrices.
  • Les quatre centres remarquables sont confondus.
Médiane

Une médiane est une droite passant par un sommet et le milieu du côté opposé.

Médiatrice

La médiatrice d'un segment est la droite perpendiculaire à ce segment passant par son milieu.

Hauteur

Une hauteur est une droite passant par un sommet et perpendiculaire au côté opposé.

Centre de gravité

Le centre de gravité est le point d'intersection des trois médianes.

On le note généralement \(G\).

Orthocentre

L'orthocentre est le point d'intersection des trois hauteurs.

XII. Le conjugué d'un nombre complexe

Définition

Soit :

\[ z=a+ib \]

Le conjugué de \(z\) est :

\[ \boxed{ \overline z=a-ib } \]
Exemple
\[ z=4+3i \] \[ \overline z=4-3i \]
Interprétation géométrique

Le point d'affixe \(\overline z\) est le symétrique du point d'affixe \(z\) par rapport à l'axe réel.

XIII. Exercice d'approfondissement

Énoncé

Soit :

\[ j= \frac{-1+i\sqrt3}{2} \]

Dans le plan complexe, on considère les points d'affixes :

\[ A(1) \] \[ B(j) \] \[ C(j^2) \]

Montrer que le triangle \(ABC\) est équilatéral.

Correction

Calculons le module de \(j\).

\[ |j| = \sqrt{ \left( -\frac12 \right)^2 + \left( \frac{\sqrt3}{2} \right)^2 } \] \[ |j| = \sqrt{ \frac14+\frac34 } \] \[ |j|=1 \]

Les points d'affixes \(1\), \(j\) et \(j^2\) appartiennent donc au cercle trigonométrique.

On sait également que :

\[ j^3=1 \] \[ 1+j+j^2=0 \]

Les trois racines cubiques de l'unité sont réparties régulièrement sur le cercle.

L'angle au centre entre deux sommets consécutifs vaut :

\[ 120^\circ \]

Les trois côtés sont donc de même longueur.

\[ AB=BC=CA \]

Ainsi :

\[ \boxed{ ABC \text{ est un triangle équilatéral} } \]
À retenir
\[ \overline z=a-ib \] \[ j^3=1 \] \[ 1+j+j^2=0 \]

Les images des racines cubiques de l'unité forment un triangle équilatéral.

Dans un triangle équilatéral :

  • Les médianes sont aussi des hauteurs.
  • Les médiatrices sont aussi des médianes.
  • Le centre de gravité et l'orthocentre sont confondus.
